JobCube から新規に構築したJobCube2へデータをインポートします。

【JobCubeでのデータ準備】
1. JobCube の tool.php を開き、
　ページ下部にある「一括エクスポート」のリンクをクリックしてください。
　各テーブルのデータが [JobCubeの設置先]/db/tdb/ へファイル出力されます。
※tdbフォルダ、csvファイルのパーミッションが「777」になっている必要があります。
csvファイルにデータが存在しない場合はパーミッションを確認してからもう一度「一括エクスポート」を実行してください。

2. JobCube の db フォルダをコピーして import という名前に変更して JobCube2 に配置してください。
※dbフォルダをコピーせずに名前を変更するとJobCubeが動作しなくなります。


【JobCube2でデータの取り込み】
1. tool.php へログイン
　インストール手順を完了させてください。

2. 「◆その他の操作」の項目の「パッチの適用」のボタンを押し、
　「jc1tojc2」の欄の「適用」ボタンを押してください。

3. 準備ができましたら、
　「適用する」のボタンを押してください。」の表示が出ますので、
　「適用する」のボタンを押してください。

4. エラーが出ずに終了すれば完了です。
　エラーが出た場合は、JobCube のデータが正しく配置されているかご確認ください。
